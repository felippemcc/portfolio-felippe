# Portfolio — Felippe Moura

Portfólio pessoal com projetos de análise de dados, automação e BI.

## Estrutura
- `index.html` — página principal
- `css/style.css` — estilos
- `js/main.js` — scripts
- `assets/images/` — imagens e foto de perfil

## Como rodar localmente
1. `git clone https://github.com/<USERNAME>/portfolio-felippe.git`
2. `cd portfolio-felippe`
3. Abra `index.html` no navegador (ou `Live Server` no VS Code).

## Publicação
Usamos GitHub Pages para publicar a versão estática.
